# Calibration Uncertainty Calculator

A Progressive Web App for calculating, validating, and generating reports for measurement uncertainties in industrial calibration processes. Built for Instruworks LLC FZ, compliant with ISO/IEC 17025:2017.

## Stack
- Frontend: React (hosted on Vercel)
- Backend: Python FastAPI (hosted on Railway)
- Database: Supabase (PostgreSQL)
- PDF Reports: ReportLab
- Excel Reports: openpyxl
